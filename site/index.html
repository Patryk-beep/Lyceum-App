What it is

A portable, evidence-based way to actually learn things.

Lyceum runs a complete learning loop built on the techniques with the strongest empirical support. The app makes that loop interactive and standalone — same science, purpose-built interface.

Lyceum is subject-agnostic. Point it at calculus, the cello, contract law, or Rust, and it researches the domain into a teachable knowledge map, places you at the right level, and builds a backward-designed curriculum ordered by a prerequisite graph.

From there it teaches concrete-first, checks your understanding, gives you assignments calibrated to the edge of your ability, grades them with feedback that teaches, schedules spaced reviews, and finishes with a defended capstone and certification.

The companion app takes all of this out of a chat transcript and into a dedicated learning environment — same science, same loop, purpose-built interface.

Built on learning science

The methods with the strongest empirical support:

  • Retrieval practice
  • Spaced repetition
  • Interleaving
  • Deliberate practice
  • Mastery learning
  • Backward design
  • ✓Mastery gating — you advance only when you’ve actually demonstrated competence.
  • ✓Durable memory — recall, then review on a schedule, so learning sticks.
The learning loop

Nine coordinated skills, in order.

Each step reads the course state, does its job, and writes the state back — so the loop is resumable, transferable, and honest about what you’ve mastered.

01

learn

The orchestrator and entry point — sets up the workspace, captures your start and target level, and routes to the next step.

02

research-topic

Deep research into a teachable knowledge map: concepts, prerequisites, misconceptions, level descriptors, and authentic tasks.

03

placement-test

A roughly ten-item adaptive test that recommends the right level for you to start from.

04

build-curriculum

A backward-designed curriculum across your level band, ordered by a prerequisite graph.

05

teach-lesson

Teaches the current module concrete-first, checks for understanding, closes with free recall, and seeds spaced reviews.

06

create-assignment

A level-appropriate assignment and rubric, calibrated to the edge of your ability.

07

assess-understanding

Grades a submission, gives feedback that teaches, updates mastery, and schedules reviews — the tracking engine.

08

review-session

Spaced-repetition review of everything that’s due — the biggest long-term-retention lever in the loop.

09

capstone

An authentic, defended, rubric-scored mastery project — and certification.

research→ placement→ curriculum→ teach→ assign→ assess→ review→ capstone

It’s a cycle, not a checklist: steps 05–08 (teach → assign → assess → review) repeat per module, with assessment and review feeding back into teaching until you reach mastery — then the course closes at the capstone.

How it works

Designed so your progress can’t lie or vanish.

A few architectural ideas keep Lyceum durable, honest, and yours.

State, not conversation

Every step reads manifest.json, acts, and writes it back. Routing is computed from disk — so a course can pause, resume across sessions, and survive a context reset.

Your folder is the save file

All course state lives in plain files you can copy anywhere. Move a course between machines by copying a folder — no account, no lock-in, no database to lose.

Single writer for mastery

Only assessment and review write mastery scores, so “feeling fluent” never inflates the data. Domain behavior is data in the curriculum, not hardcoded logic — fully subject-agnostic.

Available now

Use Lyceum today, inside Claude Code.

The app is still in development — but the Lyceum plugin that powers it is ready to use right now. Install it with two commands.

Plugin · stable & open source (MIT)

To be clear: the Lyceum App described on this page is in development and is not yet installable. The commands below install the Lyceum plugin — the working system the app is built on — into Claude Code today.

Claude Code 
# 1 · add the Lyceum marketplace
/plugin marketplace add Patryk-beep/lyceum

# 2 · install the plugin
/plugin install lyceum@lyceum-marketplace

Run these in Claude Code, then start learning with the learn skill.

Plugin repository Get Claude Code
  1. Install Claude Code if you haven’t — it’s the host the plugin runs in. code.claude.com
  2. Add the marketplace and install Lyceum with the two commands on the left.
  3. Start a course — invoke learn, name your subject, and pick your target level.
  4. Keep your folder. Your manifest.json course state travels with you — pause and resume any time.
